# WORKFLOW - TASK_CENTER

## Trạng thái
- NEW
- ASSIGNED
- IN_PROGRESS
- WAITING
- DONE
- CANCELLED
- ARCHIVED

## Transition
- NEW -> ASSIGNED
- ASSIGNED -> IN_PROGRESS
- IN_PROGRESS -> WAITING
- WAITING -> IN_PROGRESS
- IN_PROGRESS -> DONE
- WAITING -> CANCELLED
- NEW -> CANCELLED
- DONE -> ARCHIVED
