# CBV LOG STANDARD

## Mọi log phải có
- ID
- MODULE
- ENTITY_TYPE
- ENTITY_ID
- ACTION
- BEFORE_JSON
- AFTER_JSON
- NOTE
- ACTOR_ID
- CREATED_AT

## Action gợi ý
- CREATED
- UPDATED
- STATUS_CHANGED
- OWNER_CHANGED
- CHECKLIST_ADDED
- CHECKLIST_DONE
- ATTACHED_FILE
- CONFIRMED
- CANCELLED
- ARCHIVED

## Mục tiêu
- Truy được ai làm gì
- Khôi phục được bối cảnh
- Audit được hành vi hệ thống
