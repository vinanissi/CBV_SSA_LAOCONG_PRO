## PART 3 — AppSheet Formula Suggestions

### Show_If = FALSE (always hidden)

- ID, CREATED_AT, CREATED_BY, UPDATED_AT, UPDATED_BY, IS_DELETED, BEFORE_JSON, AFTER_JSON, CONFIRMED_AT, CONFIRMED_BY

- All STATUS fields (change via GAS actions)
- All audit fields
- TASK_UPDATE_LOG, FINANCE_LOG: all columns
- ADMIN_AUDIT_LOG: all columns

```
FINANCE_TRANSACTION.TRANS_CODE:    [STATUS] <> "CONFIRMED"

FINANCE_TRANSACTION.AMOUNT:        [STATUS] <> "CONFIRMED"

FINANCE_TRANSACTION.PAYMENT_METHOD:[STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.DON_VI_ID:     [STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.COUNTERPARTY:  [STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.REFERENCE_NO:  [STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.RELATED_*:     [STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.DESCRIPTION:   [STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.EVIDENCE_URL:  [STATUS] <> "CONFIRMED"
FINANCE_TRANSACTION.IS_STARRED:    TRUE
FINANCE_TRANSACTION.IS_PINNED:   TRUE
```

### Valid_If (enum-backed)

See 04_APPSHEET/APPSHEET_ENUM_BINDING.md. Example:

```

Editable_If: AND([IS_SYSTEM] = FALSE, [ALLOW_EDIT] = TRUE)
```
