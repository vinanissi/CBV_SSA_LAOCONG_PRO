# APPSHEET SECURITY FILTERS

**Note:** User filters use USER_DIRECTORY.ID. OWNER_ID and REPORTER_ID store USER_DIRECTORY.ID, not email.

## HO_SO_MASTER
- ADMIN: `TRUE`
- OPERATOR: `OR(ISBLANK([OWNER_ID]), [OWNER_ID] = FIRST(SELECT(USER_DIRECTORY[ID], AND([STATUS] = "ACTIVE", LOWER([EMAIL]) = LOWER(USEREMAIL())))))`
- VIEWER: `TRUE`

## TASK_MAIN
- ADMIN: `TRUE`
- OPERATOR: `OR([OWNER_ID] = FIRST(SELECT(USER_DIRECTORY[ID], AND([STATUS] = "ACTIVE", LOWER([EMAIL]) = LOWER(USEREMAIL())))), [REPORTER_ID] = FIRST(SELECT(USER_DIRECTORY[ID], AND([STATUS] = "ACTIVE", LOWER([EMAIL]) = LOWER(USEREMAIL())))))`
- VIEWER: `TRUE`

## FINANCE_TRANSACTION
- ADMIN: `TRUE`
- OPERATOR: `TRUE`
- VIEWER: `TRUE`

Ghi chú: nếu sau này có UNIT_ID chuẩn thì thay bằng filter theo đơn vị.
